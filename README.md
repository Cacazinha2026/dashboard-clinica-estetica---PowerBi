# dashboard-clinica-estetica---PowerBi
Dashboard Clínica Estética — Power BI

Solução de Business Intelligence desenvolvida em Power BI para gestão estratégica de clínica estética.

O projeto surgiu a partir de uma situação real observada em uma clínica que realizava o controle de sessões manualmente, utilizando papéis e registros descentralizados. A proposta foi criar uma solução piloto capaz de transformar dados operacionais em insights estratégicos para apoio à tomada de decisão.

📊 Funcionalidades do Dashboard
Visão geral da operação
Controle de sessões e recorrência
Gestão financeira
Indicadores de clientes
Performance de procedimentos
Indicadores de profissionais
Relatórios executivos
💡 Principais Indicadores
Receita total
Ticket médio
Taxa de conversão
Receita perdida
Retenção de clientes
Curva ABC
Performance operacional
🛠 Tecnologias Utilizadas
Power BI
DAX
Power Query
Modelagem de Dados
UX/UI para Dashboards
